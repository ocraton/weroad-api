Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

GET api/user

Example request:
curl --request GET \
    --get "http://localhost/api/user" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/user"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

GET api/travels/{travel_slug}/tours

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

URL Parameters

travel_slug   string   

The slug of the travel. Example: iceland-hunting-northern-lights

Body Parameters

priceFrom   number  optional  

Example: 76414.993993

priceTo   number  optional  

Example: 1.7260321

dateFrom   string  optional  

Must be a valid date. Example: 2023-11-13T15:36:38

dateTo   string  optional  

Must be a valid date. Example: 2023-11-13T15:36:38

sortBy   string  optional  

Example: endingDate

Must be one of:
  • name
  • startingDate
  • endingDate
  • price
orderBy   string  optional  

Example: desc

Must be one of:
  • asc
  • desc
